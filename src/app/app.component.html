Definición de Módulos

son contenedores que te permiten agrupar componentes y servicios que formen parte de una aplicación, de esta manera se trabaja de una forma organizada, además Angular maneja el concepto de árbol para la jerarquía de módulos En angular el modulo raíz es el AppModule que se genera autoḿticamente al crear un proyecto

Definición de Componente

Los componentes son almacenados en los módulos, estos componentes sirven para creaciones de páginas web en angular se compone de dos partes principales la vista y el controlador Cuando se crea un proyecto en angular el componente que se crea por default es el app.component.ts, que no es más que una clase ts

Definición de Directivas

Se usan para añadir, modificar y eliminar características de elementos DOM en el html,se aplican a los elementos padres (huésped) y esta a su vez se extiende a los elementos hijos, las directivas son precedidas del *

Definición de Pipes

Permiten transformar visualmente la información, por ejemplo, cambiar un texto a mayúsculas o minúsculas, o darle formato de fecha y hora.

Estructura de una Aplicacion

Conocer la estructura del proyecto ayuda a tranajar de forma ordenada, se detalla las principales carpetas

  1. node_modules: Contiene todas las dependencias de nuestro proyecto.
  2. src: Directorio que contiene los módulos, es el más importante, ya que contiene todo el código.
    • Carpeta app: Se ubica toda la implementación de los componentes principales, junto a su template html y archivos de estilos css.
    • Carpeta asset: Contendrá todos los asset y archivos adicionales para hacer que el proyecto funcione.
    • Carpeta enviroments: Se encuentra las configuraciones y variables de entorno para poner el proyecto en desarrollo o producción.
    • Archivo index.html: Es el archivo de la paguina proncipla del proyecto
  3. package.json: es la configuración de nuestra aplicación.
  4. tsconfig.json: contiene la configuración TypeScript.
